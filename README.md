# devops-course
База знаний теоретической информации о практиках и инструментах DevOps и практические задания для начинающих.
Данный курс разделён на модули. В каждом модуле представлена краткая теоретическая информация, практические задания, примеры, ссылки на полезные ресурсы и чек-листы с вопросами для самопроверки усвоения материала.
Данный курс состоит из следующих модулей:
1. git
2. linux
3. Docker
4. K8s
5. Jenkins
6. Ansible
7. Aws
8. Terraform
